NSI : Séance S40 — JavaScript (1)

Thème : Syntaxe de base et Manipulation du DOM

1. Variables et Types de données

En JavaScript moderne, on utilise principalement let (variable) et const (constante).

let temperature = 21.5; // Nombre
const unite = "°C"; // Chaîne de caractères
temperature = temperature + 1; // Modification possible

2. Accéder au DOM (Document Object Model)

Pour modifier un élément, il faut d'abord le "trouver" dans la page grâce à son identifiant :

const monTitre = document.getElementById("id-de-l-element");

3. Modifier le contenu et le style

Une fois l'élément sélectionné, on peut agir sur ses propriétés :

Action Code JavaScript Résultat
Modifier le texte elt.textContent = "Nouveau texte"; Remplace le texte brut.
Modifier le style elt.style.color = "red"; Applique du CSS en ligne.
Cacher l'élément elt.style.display = "none"; L'élément disparaît de la page.

4. Les Fonctions et Événements

On regroupe souvent les instructions dans une fonction pour les déclencher lors d'un clic.

function alerteTemp() {
    alert("Température trop élevée !");
}

Dans le HTML, on lie la fonction au bouton avec l'attribut onclick :

<button onclick="alerteTemp()">Vérifier</button>
