Exercice 1 : Compréhension et Vocabulaire

Manipulation de base d'un dictionnaire d'inventaire :

  • Type : La variable inventaire est de type dict (dictionnaire).
  • Terminologie : "pieces" est la clé et 150 est la valeur.
  • Accès : print(inventaire["potions"]).
  • Mise à jour : inventaire["potions"] = 6.
  • Ajout : inventaire["bouclier"] = 1.

Exercice 2 : Parcours et Filtrage

Pour filtrer les données d'un dictionnaire, on utilise la méthode .items() qui permet d'extraire simultanément les clés et les valeurs :

for produit, prix in catalogue.items():
    if prix < 30:
        print(produit)

Note : Le script parcourt chaque couple, teste si le prix est inférieur à 30, et affiche la clé correspondante.

Exercice 3 : Algorithme de Fréquences

L'algorithme de calcul d'occurrences est un classique. Voici le code complété :

for caractere in texte:
    if caractere in frequences.keys():
        frequences[caractere] = frequences[caractere] + 1
    else:
        frequences[caractere] = 1

Explication : On teste si le caractère est déjà une clé du dictionnaire. Si oui, on incrémente son compteur ; sinon, on crée la clé avec la valeur initiale 1.

def est_dispo(titre_recherche, table_livres):
