NSI : Séance WEB01 — Langages du Web

Thème : Interactions Homme-Machine (IHM) sur le Web

1. Les Fondamentaux

  • Langage de description : Le HTML5 décrit la hiérarchie des données (DOM).
  • Balises : Doubles (<p>...</p>) ou orphelines (<img>).

2. Récapitulatif des balises courantes

Balise Rôle Type
<h1> à <h6> Titres (h1 : titre principal, h2 : section...) Double
<p> Paragraphe de texte Double
<ul> Liste non-ordonnée (à puces) Double
<ol> Liste ordonnée (numérotée) Double
<li> Élément d'une liste (List Item) Double
<button> Composant d'interaction (bouton) Double
<img> Insertion d'une image Orpheline

3. Syntaxe des éléments complexes

A. Ajouter un lien hypertexte :

<a href="https://www.wikipedia.fr">Visiter Wikipédia</a>

B. Insérer une image :

<img src="image.jpg" alt="Description de l'image">

C. Les listes (ul vs ol) :

Liste à puces (ul) :
<ul>
  <li>Élément A</li>
  <li>Élément B</li>
</ul>
Liste numérotée (ol) :
<ol>
  <li>Étape 1</li>
  <li>Étape 2</li>
</ol>

4. Structure minimale d'un document

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Dashboard Domotique</title>
</head>
<body>
    <h1>Contrôle du Salon</h1>
    <p>Bienvenue sur votre interface.</p>
</body>
</html>
