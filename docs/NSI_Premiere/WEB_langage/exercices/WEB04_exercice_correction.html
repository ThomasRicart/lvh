Correction WEB04 - JavaScript : Syntaxe et Interactivité

Exercice 1 : Analyse de code et Syntaxe JS

  • Variable (peut changer) : intensite (déclarée avec let).
  • Constante (ne change pas) : piece (déclarée avec const).
  • Type de données de 'piece' : Chaîne de caractères (String).
  • Nom de la fonction : augmenterLumiere.

Exercice 2 : Manipulation du DOM

Le DOM (Document Object Model) permet de relier le script aux éléments de la page :

Instruction Action correspondante
document.getElementById("id") B. Cible un élément précis de la page via son identifiant.
.textContent = "..." C. Change le texte affiché à l'intérieur de l'élément.
.style.color = "..." A. Modifie la couleur du texte de l'élément.

Exercice 3 : Événements et Interactivité

1. Côté HTML (Appel de la fonction) :

<button onclick="direBonjour()">Connexion</button>

2. Côté JavaScript (Modification du texte) :

function direBonjour() {
    const affichage = document.getElementById("message");
    affichage.textContent = "Systeme Pret";
}

Exercice 4 : Application au Dashboard

Code complet pour activer le "Mode Nuit" dynamiquement :

function activerModeNuit() {
    // 1. Recuperer l'element d'ID "dashboard-bg"
    const ecran = document.getElementById("dashboard-bg");

    // 2. Changer la couleur de fond (backgroundColor) en "black"
    ecran.style.backgroundColor = "black";

    // 3. Changer la couleur du texte (color) en "white"
    ecran.style.color = "white";
}
