Exercice 1 : Anatomie d'un formulaire

  • Méthode d'envoi utilisée : La méthode GET[cite: 38].
  • Fichier de réception : Les données seront envoyées vers le fichier reglage.html[cite: 39].
  • Rôle de l'attribut name="v_temp" : Cet attribut définit le nom de la variable qui contiendra la valeur saisie. C'est l'identifiant que le serveur utilisera pour récupérer la donnée[cite: 40].

Exercice 2 : Analyse d'URL (Méthode GET)

1. URL complétée après validation (valeur 22) :

http://monserveur.fr/reglage.html?v_temp=22

2. Problème de sécurité du GET pour un mot de passe :

Les données envoyées en méthode GET transitent directement dans l'URL. Si un mot de passe est saisi, il apparaît en clair dans la barre d'adresse et dans l'historique du navigateur, ce qui est une faille de sécurité majeure[cite: 42, 43].
