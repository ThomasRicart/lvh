NSI : Séance WEB06 — IHM & Objets Connectés

Thème : Capteurs, Actionneurs et Chaîne d'Information

1. La Chaîne d'Information

Tout système informatique connecté repose sur une boucle logique composée de trois maillons :

CAPTEUR
(Entrée / Input)
➔
TRAITEMENT
(Algorithme JS)
➔
ACTIONNEUR
(Sortie / Output)

2. Définitions clés

  • Capteur : Dispositif qui transforme une grandeur physique (température, luminosité, clic) en une donnée numérique utilisable par le programme.
  • Actionneur : Composant qui exécute une action physique ou visuelle (allumer une LED, faire tourner un moteur, afficher un message d'alerte).
  • IHM (Interface Homme-Machine) : Le point de rencontre entre l'utilisateur et le système (notre Dashboard HTML).

3. Interaction Bidirectionnelle

Une IHM ne sert pas qu'à lire des données, elle permet aussi de piloter le système :

function modeAuto() {
    // L'IHM force la valeur du capteur
    document.getElementById("consigne").value = 21;
    // On relance le traitement
    mettreAJourDashboard();
}
